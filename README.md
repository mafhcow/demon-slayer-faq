# Demon Slayer FAQ

A comprehensive, responsive FAQ and guide for the Demon Slayer class in MapleStory.

## Features
- **Responsive Design**: optimized for desktop, tablet, and mobile viewports.
- **Dynamic Content**: loads FAQ sections from Markdown files.
- **Searchable**: easily find information about skills, gearing, and mechanics.

## Local Development
To run the project locally, you can use a simple HTTP server. For example, using Python:
```bash
python -m http.server
```
Then visit `http://localhost:8000` in your browser.

## Built With
- HTML5 & Semantic CSS
- JavaScript (Vanilla)
- [Marked.js](https://marked.js.org/) for Markdown parsing
- [Inter Font](https://fonts.google.com/specimen/Inter)
